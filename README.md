Tongtsz
================

Tongtsz is a Cantonese word repository. Tongtsz is the pronunciation of 唐字.

Ruby on Rails
-------------

This application requires:

- Ruby 2.5.1
- Rails 4.2.10

Learn more about [Installing Rails](http://railsapps.github.io/installing-rails.html).
